# OIBSIP_python_3
Random Password Generator(Task 3)

# Objective
Create a Random Password Generator in python that generates secure passwords based on user-defined criteria such as length and character types(numbers, letters, symbols).

# Project Level
--**Beginner version**: Command line interface(CLI)

# Steps Performed
- User can specify password length
- Options to include/exclude: numbers, letters, symbols
- Ensures at least one character from each selected category

 # Tools and libraries
 - Python 3.x
 - Standard libraries: 'random', 'string'

# Outcome
Successfully built a secure and customizable password generator that helps users create strong passwords easily.
